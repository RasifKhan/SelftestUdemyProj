----------------------------------------------------------

First create a folder >  AngularEcom

-----------------------

F:\Dot Net Practice 2025\Angular Course Udemy\AngularEcom>dotnet new sln

F:\Dot Net Practice 2025\Angular Course Udemy\AngularEcom>dotnet new webapi -o API -controllers

F:\Dot Net Practice 2025\Angular Course Udemy\AngularEcom>dotnet new classlib -o Infrastructure

-----------------------------------

F:\Dot Net Practice 2025\Angular Course Udemy\AngularEcom>dotnet sln add API

F:\Dot Net Practice 2025\Angular Course Udemy\AngularEcom>dotnet sln add Core

F:\Dot Net Practice 2025\Angular Course Udemy\AngularEcom>dotnet sln add Infrastructure


then check the soloution list 
----------------------------------

------------------------


F:\Dot Net Practice 2025\Angular Course Udemy\AngularEcom>cd API

F:\Dot Net Practice 2025\Angular Course Udemy\AngularEcom\infrastructure>dotnet add reference ../Core
